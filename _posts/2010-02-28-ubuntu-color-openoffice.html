I like the colors chosen by Ubuntu and I tend to use them for my presentations and posters. So, I created a palette that can be imported in OpenOffice. How to use it?

  1. Download the palette
  2. In OpenOffice, go in Format > Area for instance
  3. Click on the "Colors" tab
  4. Click on the folder icon ("Load Color list")
  5. Browse for the file you downloaded
  6. Enjoy, Ubuntu colors within OpenOffice
